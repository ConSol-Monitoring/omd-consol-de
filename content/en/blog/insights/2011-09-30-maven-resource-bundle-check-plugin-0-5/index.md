The new <a href="http://labs.consol.de/projects/maven/maven-rbc-plugin">maven-rbc-plugin</a> 0.5 release contains minor bugfixes.

<!--more-->
The most notable changes are:
<ul>
	<li>Report lists locales for summary and per bundle</li>
	<li>Report now shows number of issues in the summary</li>
</ul>
See the <a href="http://labs.consol.de/projects/maven/maven-rbc-plugin/changes-report.html#a0.5">changelog</a> for details, and the <a href="http://labs.consol.de/projects/maven/maven-rbc-plugin/usage.html">usage instructions</a> for configuring the plugin.
